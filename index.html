Webscraping using Python BeautifulSoup

In this scraped I have used BeautifulSoup4 (BS4) which is a framework that is built into Python. BS4 is a great frame that allows users to grab content from the web and store that content. For the base of this project, I used BS4 to grab text-based content from engineeringnz.org and cleaned and stored the content into rows and columns in a table format using Python's Pandas. This project was conducted with a group of 4 people and I was in charge of scraping the content.
